# Scope

- node: docs/foundation/scope.md
	- in_scope:
		- runtime JSON config loading
		- command surface tied to the achievement catalog
		- GUI menus for the achievement catalog
		- immersive large-scale MMORPG atmosphere across the full experience
		- menu presentation that matches the world tone and ritual aesthetic
		- canonical authority is the source tree plus this spec tree
	- out_of_scope:
		- skill systems
		- player storage or progress persistence
	- non_authoritative_artifacts:
		- build/ outputs
		- generated sources and resources
		- data/ runtime state

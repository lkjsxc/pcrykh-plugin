# Interaction

This directory defines all player-facing interaction surfaces.

## TOC

- [commands/README.md](commands/README.md)
- [gui/README.md](gui/README.md)
- [chat.md](chat.md)

## Orientation

- Commands define the textual control surface.
- GUI defines the primary player experience.
- Chat defines global announcement and tip behavior.

## Entry points

- `/pcrykh` is the root command.
- The hotbar item is the primary GUI launcher.

# Style guide

## Theme goals

- Bright highlights, informative lore, strong iconography, and consistent click sounds.

## Layout conventions

- 54-slot inventories are preferred for menus.
- Top row reserved for title/summary items.
- Bottom row reserved for navigation:
  - Slot 45: Back
  - Slot 49: Close
  - Slot 47: Previous Page
  - Slot 51: Page Indicator
  - Slot 53: Next Page

## Iconography

- Achievements MUST have a unique icon material.
- Locked items MUST show a consistent “locked” lore block.

## Feedback

- Successful clicks: a pleasant UI sound.
- Denied clicks: a dull/error UI sound.

### Action bar feedback

- AP awards MUST emit an action bar message (see [gui/action-bar/README.md](../action-bar/README.md)).

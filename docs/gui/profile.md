# Profile page

- node: docs/gui/profile.md
  - purpose:
    - provide the player's achievement summary and identity badge
  - inventory:
    - size: 54 slots
    - title: `Profile`
    - all clicks are cancelled; no item movement is permitted
  - layout:
    - slot `20`: player identity
    - slot `24`: achievement summary
    - slot `45`: back button
  - item_rendering:
    - identity:
      - material: `PLAYER_HEAD`
      - name: `You`
      - lore:
        - `name: <player_name>`
        - `uuid: <player_uuid>`
    - summary:
      - material: `PAPER`
      - name: `Progress`
      - lore:
        - `completed: <completed>/<total>`
        - `ap: <ap_total>`
        - `completion: <percent>%`
    - back:
      - material: `BARRIER`
      - name: `Back`
  - behavior:
    - clicking `Back` opens [menu.md](menu.md)

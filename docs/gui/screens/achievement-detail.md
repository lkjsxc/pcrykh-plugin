# Screen: Achievement Detail

## Purpose

View and complete progression within one achievement.

## Required elements

- Header summary:
  - Achievement name
  - Current tier
  - Current objective
- “Objective” item:
  - Description
  - Completion criteria
  - Rewards (AP)
- History item:
  - Last completed levels (at least 5)

## Completion model

- Objectives are completed by gameplay events (not by clicking).
- The GUI MUST reflect completion in real time or via refresh.

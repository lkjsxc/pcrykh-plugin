# GUI

This directory defines the GUI system. The GUI MUST evoke a dense, polished RPG menu aesthetic (dense menus, polished item icons, clear navigation, satisfying feedback).

## Global GUI invariants

- GUIs MUST be inventory-based.
- GUIs MUST be navigable without typing.
- Every GUI MUST have:
  - A “Back” button (except main menu)
  - A “Close” button
  - A consistent footer layout
